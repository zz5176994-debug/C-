## 一、:heart:**逻辑运算** 
- 逻辑运算是对逻辑量进行的运算，结果只有0或1 
- 逻辑量是关系运算或逻辑运算的结果
  
| 运算符 | 描述   | 示例     | 结果说明 |
|--------|--------|----------|----------|
| `!`    | 逻辑非 | `!a`     | 如果 a 是 true，结果是 false；如果 a 是 false，结果是 true |
| `&&`   | 逻辑与 | `a && b` | 如果 a 和 b 都是 true，结果是 true；否则是 false |
| `\|\|`  | 逻辑或 | `a \|\| b` | 如果 a 和 b 有一个是 true，结果为 true；两个都是 false，结果为 false |  

*4.3.* 例子:thinking:

```c
/*如何判断一个字符c是否为大写字母*/
c >= 'A' && c <= 'Z'
```
理解一下 :thinking: 
```c
!age<20  /*!是单目运算符，单目运算符的优先级要高于双目运算符
所以先做！运算（如果age=0，那么！age=1；反之！age为1）
再去和20比较
*/
//正确做法
!(age<20)
```

*4.5.* 短路
逻辑运算是自左向右进行的，如果左边的结果已经能够决定结果了，就不会做右边的计算
> a == 6 && b == 1
> a == 6 && b += 1

//对于&&,左边是false时就不做右边了
对于||，左边是true时就不做右边了
:fire:所以不要把赋值或者复合赋值写进表达式  

## 二、条件运算  

*1.* **形式**：  
`count = (count > 20)?count - 10:count + 10`  

- ?前面是条件；?后面是满足条件时的值
- ：后面是条件不满足时的值
相当于  

```c  
if(count>20)
    count = count - 10;
else
    count = count + 10;
```  
  
*2.* **优先级**

- 条件运算符的优先级高于赋值运算符，但是低于其他运算符  

理解一下:thinking:  

```c
m<n ?x: a+5
a++ >= 1 && b-- > 2? a: b 
x=3 * a > 5 ? 5 :20
```  

## 三、逗号运算
*1.* **形式**： 
- 逗号用来连接两个表达式,并以其右边的表达式的值作为它的结果。  

*2.* **优先级**
- 逗号的优先级是所有的运算符中最低的，所以它两边的表达式会先计算；逗号的组合关系是自左向右，所以左边的表达式会先计算，而右边的表达式的值就留下来作为逗号运算的结果。  

*3.* **应用**  
- 主要在for表达式中用到
`for(i = 0,j = 10;i < j;i++,j--)`
